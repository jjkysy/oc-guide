# 字节跳动 — 扣子 OpenClaw

> [返回目录](README.md)

## 简介

字节跳动通过扣子（Coze）平台深度集成 OpenClaw，推出"扣子 OpenClaw"，是目前国内零门槛部署 OpenClaw 最友好的平台之一。无需服务器、不用命令行，5 分钟即可拥有可执行任务的 AI 助手，可一键发布到飞书、微信、钉钉等 IM。同时扣子编程（原 Coze 开发平台）支持一键部署 OpenClaw 项目。

## 产品形态

| 产品 | 说明 |
|------|------|
| **扣子 OpenClaw** | 可视化工作流 AI Agent，深度集成豆包大模型，拖拽定制任务流程 |
| **扣子编程** | 支持一键部署 OpenClaw 项目，创建的 Skills 可加载到 OpenClaw |
| **豆包 2.0 API** | Pro/Lite/Mini 三款 Agent 模型 + Code 版，价格极低 |

## 特点与优劣

**优势：**
- 零代码零服务器，5 分钟完成部署
- 可视化拖拽工作流，定制 Agent 门槛极低
- 免费用户可部署最多 2 个实例，云端 7×24 小时运行
- 深度集成豆包 2.0 系列模型，API 价格"比海外模型低了一个数量级"
- 支持发布到飞书、微信、钉钉等多平台
- 扣子编程支持一键部署，Skills 可复用

**劣势：**
- 免费版实例有资源上限，复杂任务会被限流
- 对原生 ClawHub 技能的兼容度一般
- 数据在云端，不适合对隐私要求极高的场景
- 非开源，自定义深度有限
