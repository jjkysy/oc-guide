# 腾讯 — WorkBuddy / QClaw

> [返回目录](README.md)

## 简介

腾讯在 2026 年 3 月推出双线 OpenClaw 产品：面向 B 端企业的 **WorkBuddy** 和面向 C 端个人用户的 **QClaw**。WorkBuddy 基于腾讯 CodeBuddy 同源架构，深度集成企业微信；QClaw 独家打通个人微信和 QQ，是国内唯一支持微信直连的 OpenClaw 变体。

## 产品形态

| 产品 | 定位 | 说明 |
|------|------|------|
| **WorkBuddy** | B 端企业 | 免部署桌面应用，兼容 OpenClaw Skills，企微远程遥控，内置 20+ Skills 技能包与 MCP 协议，支持安全审计 |
| **WorkBuddy Claw** | B 端 IM Agent | 国内首个 IM 原生 AI Agent 平台，在企微/飞书/钉钉中 @即用 |
| **QClaw** | C 端个人 | 本地部署，数据不上云，打通个人微信 + QQ 双端，接入 5000+ OpenClaw 社区技能 |

## 特点与优劣

**优势：**
- WorkBuddy 安装即用，最快 1 分钟完成企微配置
- QClaw 独家打通个人微信对话，杀手级功能
- 完全兼容 OpenClaw 技能生态，Skills 可无缝迁移
- 基于 CodeBuddy 架构，统一账号与计费，企业级安全审计
- 腾讯内部超 2000 名非技术员工已在使用，经过实战验证
- 新用户无门槛赠送 5,000 Credits

**劣势：**
- WorkBuddy 非开源，自定义灵活度有限
- QClaw 的微信直连实际通过企微"微信客服"入口，有一定限制
- 依赖腾讯生态，跨生态协作受限
- 目前仍处于早期阶段，功能和稳定性可能不如原版 OpenClaw
