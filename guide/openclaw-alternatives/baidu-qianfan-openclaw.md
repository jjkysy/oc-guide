# 百度 — 千帆 OpenClaw

> [返回目录](README.md)

## 简介

百度于 2026 年 2 月将 OpenClaw 接入百度 App 搜索框，千帆平台已上架 7 个官方 Skills 至 ClawHub。百度智能云提供一键部署 OpenClaw 的云服务，用户无需自行搭建服务器。千帆平台以 Agent 为核心，为企业提供模型、Agent 开发及数据智能服务等一站式服务。

## 产品形态

| 产品 | 说明 |
|------|------|
| **百度千帆 OpenClaw** | 云端一键部署 OpenClaw 实例，集成百度智能云基础设施 |
| **百度 App 搜索集成** | 在百度 App 搜索框中直接调用 OpenClaw Agent |
| **千帆 ClawHub Skills** | 百度官方上架的 7 个 Skills（搜索、翻译、知识等） |

## 特点与优劣

**优势：**
- 百度智能云一键部署，零运维
- 百度 App 搜索入口直达，用户基数大
- 千帆平台提供完整的 Agent 开发工具链
- 与百度生态（搜索、地图、文库等）深度集成

**劣势：**
- 数据在云端，非本地部署
- 非开源，定制灵活度有限
- 自有 Skills 数量较少（7 个官方）
- IM 集成主要通过百度 App，微信/企微支持弱
