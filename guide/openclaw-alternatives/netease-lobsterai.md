# 网易有道 — LobsterAI

> [返回目录](README.md)

## 简介

LobsterAI 是网易有道于 2026 年 2 月发布的桌面 Agent 产品，支持 MCP 协议，是国内首批将 OpenClaw 与 MCP 结合的产品之一。2 月 19 日正式开源，主打教育和办公场景。

## 产品形态

| 版本 | 说明 |
|------|------|
| **LobsterAI 桌面版** | 桌面端 AI Agent，安装即用 |
| **LobsterAI 开源版** | 开源代码，可自行部署和二次开发 |

## 特点与优劣

**优势：**
- 已开源，社区可自由贡献和定制
- 支持 MCP 协议，工具扩展标准化
- 教育和办公场景优化
- 桌面端应用体验好
- 新华网等主流媒体报道，可信度高

**劣势：**
- 起步较晚，Skills 生态不如 WorkBuddy / 扣子丰富
- 社区规模较小
- IM 集成能力有限
- 企业级功能和支持较弱
