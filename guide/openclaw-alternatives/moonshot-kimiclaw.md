# 月之暗面 — KimiClaw

> [返回目录](README.md)

## 简介

KimiClaw 是月之暗面（Moonshot AI）推出的 OpenClaw 云端全托管服务，可以理解为 OpenClaw 的"精装云版本"。基于 Kimi K2.5 模型，面向付费会员开放 Beta 测试，预装 ClawHub 技能库，用户无需任何部署即可使用。K2.5 因其 Agent 能力强且价格便宜，一度成为 OpenRouter 平台上 OpenClaw 调用量最高的模型。

## 产品形态

| 版本 | 说明 |
|------|------|
| **KimiClaw** | 云端全托管 OpenClaw 服务，预装 ClawHub 技能库 |
| **Kimi K2.5 API** | 可直接在原版 OpenClaw 中接入的模型 API |

## 特点与优劣

**优势：**
- 开箱即用，完全免去部署配置流程
- 预装 ClawHub 技能库，无需手动导入
- Kimi K2.5 超长上下文能力，适合复杂长任务链
- K2.5 模型性价比高，Agent 调用量业内领先
- 兼容 OpenClaw 标准协议

**劣势：**
- 仅面向付费会员，月费约 ¥199
- 非开源，无法自定义底层行为
- IM 集成能力弱，主要通过 Web 界面使用
- 数据在云端，不适合高隐私要求场景
- 自定义 Skill 开发灵活度不如 CoPaw / 原版 OpenClaw
