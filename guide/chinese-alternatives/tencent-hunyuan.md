# 腾讯 — 混元大模型

> [返回目录](README.md)

## 简介

腾讯混元是腾讯自研的大语言模型，深度整合腾讯云生态，在微信、企业微信、腾讯会议等产品中均有应用。

## 产品形态

| 版本 | 说明 |
|------|------|
| **混元 API** | 通过腾讯云提供标准 API 接口，按 token 计费 |
| **腾讯元宝** | 面向 C 端用户的免费对话助手（网页 + App） |
| **腾讯云 TI 平台** | 企业级 AI 开发平台，支持模型微调与私有化部署 |
| **混元开源模型** | 混元系列开源权重，可本地部署 |

## 特点与优劣

**优势：**
- 腾讯生态深度整合（企业微信、腾讯文档、腾讯会议）
- 中文理解能力强，特别适合社交和内容场景
- 开源模型（混元-Large 等）支持商用许可
- 腾讯云企业客户可无缝接入

**劣势：**
- 国际化能力相对有限
- 独立开发者生态不如阿里活跃
- 多模态能力迭代节奏偏慢
