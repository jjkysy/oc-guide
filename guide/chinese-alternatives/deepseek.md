# 深度求索 — DeepSeek

> [返回目录](README.md)

## 简介

DeepSeek（深度求索）是幻方量化旗下 AI 实验室，以极低的训练成本和全面开源策略迅速崛起。DeepSeek-V3 和 DeepSeek-R1 在多项基准上达到国际顶尖水平，是目前最受关注的国产大模型之一。

## 产品形态

| 版本 | 说明 |
|------|------|
| **DeepSeek 网页/App** | C 端免费对话助手 |
| **DeepSeek API** | 开放平台 API，兼容 OpenAI 格式 |
| **DeepSeek 开源模型** | 全系列开源（V3、R1 等），支持本地和私有化部署 |

## 特点与优劣

**优势：**
- **全面开源 + MIT 许可**，商用友好度最高
- 推理能力（R1）极强，数学/代码/逻辑任务接近 GPT-4o 水平
- **API 定价极低**，性价比突出
- MoE 架构效率高，推理成本低
- 社区极度活跃，第三方适配丰富

**劣势：**
- 公司规模小，企业级 SLA 保障有限
- 不提供专属云/私有化部署服务（依赖开源自行部署）
- 多模态能力尚在发展中
- 高峰期 API 可能出现排队
