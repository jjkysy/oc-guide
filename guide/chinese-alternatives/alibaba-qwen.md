# 阿里巴巴 — 通义千问（Qwen）

> [返回目录](README.md)

## 简介

通义千问是阿里云推出的大模型系列，Qwen 开源系列在国际社区（HuggingFace）影响力极大，是国内开源大模型的标杆之一。

## 产品形态

| 版本 | 说明 |
|------|------|
| **通义千问 App** | C 端免费对话助手，支持多模态（文本、图像、文档） |
| **百炼平台** | 阿里云一站式大模型服务平台，提供 API 调用、模型微调、Agent 搭建 |
| **PAI 平台** | 阿里云机器学习平台，支持私有化训练和部署 |
| **Qwen 开源系列** | Qwen2.5 系列全量开源，覆盖 0.5B ～ 72B+ 参数规模 |

## 特点与优劣

**优势：**
- 开源生态最强，Qwen 系列在 HuggingFace 下载量长期领先
- 模型尺寸覆盖广（0.5B 到 110B+），适合各种硬件条件
- 百炼平台功能完善，API 兼容 OpenAI 格式
- 多模态能力突出（Qwen-VL 视觉模型表现优异）
- 代码能力强（Qwen-Coder 专项优化）

**劣势：**
- 企业私有化部署需要阿里云支持，独立部署文档偏少
- 产品线繁多，选型可能令新用户困惑
