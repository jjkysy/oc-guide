# 字节跳动 — 豆包（Doubao）/ 扣子（Coze）

> [返回目录](README.md)

## 简介

字节跳动旗下 AI 产品以"豆包"为 C 端入口，"火山引擎"为 B 端平台，"扣子（Coze）"为低代码 Agent 搭建平台，整体以低价策略快速抢占市场。

## 产品形态

| 版本 | 说明 |
|------|------|
| **豆包 App** | C 端免费 AI 助手（对话、写作、翻译等） |
| **火山引擎-方舟平台** | 企业级模型服务平台，提供 API 调用与模型管理 |
| **扣子（Coze）** | 低代码 AI Bot 搭建平台，支持插件、工作流、知识库 |
| **火山引擎私有化** | 面向大型企业的私有化部署方案 |

## 特点与优劣

**优势：**
- **价格极具竞争力**，API 定价是行业最低之一
- 扣子（Coze）平台零代码搭建 Agent，上手门槛极低
- 豆包 App 用户体验好，DAU 国内领先
- 强大的内容理解能力（得益于抖音/头条数据积累）

**劣势：**
- 核心模型未开源，无法本地部署
- 企业级功能与文档不如阿里、腾讯云完善
- 海外版 Coze 与国内版数据不互通
- 模型本身能力在复杂推理任务上与顶尖模型有差距
