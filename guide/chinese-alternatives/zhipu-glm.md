# 智谱 AI — GLM / ChatGLM

> [返回目录](README.md)

## 简介

智谱 AI 由清华大学技术团队创办，推出 GLM 系列模型，兼具学术基因与工程落地能力，在国内开源社区有重要影响力。

## 产品形态

| 版本 | 说明 |
|------|------|
| **智谱清言** | C 端 AI 助手 |
| **智谱开放平台** | API 服务，支持 GLM-4 系列模型调用 |
| **GLM 开源系列** | ChatGLM / GLM-4 开源权重，可本地部署 |
| **企业私有化** | MaaS 私有化部署方案 |

## 特点与优劣

**优势：**
- 清华学术背景，模型研究深度好
- GLM 开源系列在国内最早一批流行，社区基础好
- API 兼容 OpenAI 格式
- 工具调用（Function Calling）能力成熟，适合 Agent 场景

**劣势：**
- 模型规模和综合能力与阿里 Qwen 相比略有差距
- 产品矩阵不如大厂丰富
- C 端产品知名度低于豆包、Kimi
