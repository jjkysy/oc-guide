# 百度 — 文心一言（ERNIE Bot）

> [返回目录](README.md)

## 简介

百度文心大模型是国内最早发布的大模型之一，依托百度搜索和知识图谱积累，在中文理解和知识问答方面有深厚基础。

## 产品形态

| 版本 | 说明 |
|------|------|
| **文心一言 App** | C 端免费 AI 助手 |
| **千帆大模型平台** | 百度智能云的 B 端模型服务平台，支持 API 调用、微调、评估 |
| **千帆 AppBuilder** | 低代码应用搭建平台 |
| **企业私有化** | 支持在百度智能云或客户环境私有部署 |

## 特点与优劣

**优势：**
- 中文知识问答能力强，受益于百度知识图谱
- 千帆平台功能完善，企业工具链齐全
- 多模态能力覆盖（文字、图像、视频生成）
- 较早通过国内大模型备案

**劣势：**
- 核心模型未开源
- API 接口格式与 OpenAI 不兼容，迁移有一定成本
- 社区活跃度不如阿里 Qwen 系列
- 定价在国内厂商中偏高
