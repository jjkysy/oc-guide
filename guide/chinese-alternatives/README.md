## 选型建议

### 按场景推荐

| 场景 | 推荐方案 | 理由 |
|------|---------|------|
| **个人开发者 / 低成本实验** | DeepSeek API 或 Qwen 开源本地部署 | 价格最低 / 免费开源 |
| **长文档处理与分析** | Kimi | 超长上下文支持最好 |
| **企业内部知识库 / 客服** | 阿里百炼 或 腾讯混元 | 企业服务体系完善 |
| **零代码搭建 AI 应用** | 字节扣子（Coze） | 零门槛，插件丰富 |
| **私有化部署（数据合规）** | Qwen 开源 或 DeepSeek 开源 | 开源可控，许可宽松 |
| **代码辅助与开发** | DeepSeek-Coder 或 Qwen-Coder | 代码专项能力最强 |
| **已有百度云生态** | 百度千帆 | 生态整合优势 |
| **学术研究** | 智谱 GLM 或 DeepSeek | 学术合作与开源支持 |

### 按企业规模推荐

| 规模 | 推荐 | 说明 |
|------|------|------|
| 个人 / 小团队 | DeepSeek API 或 Qwen 开源 | 极致性价比 / 完全可控 |
| 中型企业 | 阿里百炼 或 字节火山引擎 | 平台能力 + 合理定价 |
| 大型企业 / 金融 / 政务 | 腾讯混元 或 百度千帆 | 完整企业服务 + 合规保障 + SLA |

---

> **注意**：以上信息截至 2025 年中。大模型行业变化极快，价格和能力排名可能随新版本发布而改变，请以各厂商官网最新信息为准。
