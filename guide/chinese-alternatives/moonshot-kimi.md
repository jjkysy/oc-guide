# 月之暗面 — Kimi

> [返回目录](README.md)

## 简介

月之暗面（Moonshot AI）是国内 AI 创业公司代表，以"长上下文"为核心卖点，Kimi 是其旗舰产品，在长文档处理和搜索增强方面表现突出。

## 产品形态

| 版本 | 说明 |
|------|------|
| **Kimi 智能助手** | C 端免费对话产品，支持超长文档上传与分析 |
| **Kimi API（Moonshot 开放平台）** | 标准 API 接口，兼容 OpenAI 格式 |
| **Kimi 企业版** | 面向企业的定制化服务 |

## 特点与优劣

**优势：**
- **超长上下文支持**（支持 200 万+ tokens），是长文档场景的首选
- 联网搜索增强效果好，适合研究和信息收集
- API 兼容 OpenAI 格式，迁移成本低
- 产品体验精致，C 端用户口碑好

**劣势：**
- 模型未开源，不支持本地部署
- 公司规模相对较小，企业级服务保障不如大厂
- 仅专注于语言模型，多模态能力有限
- 长上下文场景下延迟较高、成本较大
