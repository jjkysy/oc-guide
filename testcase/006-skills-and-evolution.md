# 测试用例：006 - 技能与进化配置

> 对应指南 [006 - 技能与进化配置](../guide/006-skills-and-evolution.md)

## TC-006-01：技能搜索与安装

**操作步骤**：
1. 执行 `openclaw skill search "calculator"`
2. 选择一个技能，执行 `openclaw skill install <name>`
3. 执行 `openclaw skill list`
4. 执行 `openclaw skill uninstall <name>`

**测试目标**：验证技能生命周期管理

**预期结果**：
- 搜索返回相关技能列表
- 安装成功，出现在 list 中
- 卸载成功，从 list 中消失

---

## TC-006-02：安全审计

**前置条件**：已安装至少一个第三方技能

**操作步骤**：
1. 执行 `openclaw security audit`
2. 查看审计报告
3. 执行 `openclaw security audit --fix`

**测试目标**：验证安全审计工具功能

**预期结果**：
- 审计报告列出检查项和结果
- `--fix` 自动修复可修复的问题
- 修复后重新审计显示改善

---

## TC-006-03：SOUL.md 自定义

**操作步骤**：
1. 编辑 `~/.openclaw/agents/default/SOUL.md`
2. 添加 "你的名字叫小助，说话简洁"
3. 重启 Gateway
4. 发送 "你叫什么名字？"

**测试目标**：验证 SOUL.md 人格配置生效

**预期结果**：
- 代理自称"小助"
- 回答风格简洁

---

## TC-006-04：MEMORY.md 长期记忆

**操作步骤**：
1. 编辑 `~/.openclaw/agents/default/MEMORY.md`，添加 "用户的猫叫咪咪"
2. 重启 Gateway
3. 发送 "我的猫叫什么？"

**测试目标**：验证长期记忆读取

**预期结果**：
- 代理回答 "咪咪"
- 无需之前的对话上下文

---

## TC-006-05：每日记忆写入

**操作步骤**：
1. 与代理进行一段包含重要信息的对话
2. 等待代理触发记忆写入（或手动请求"请保存今天的对话摘要"）
3. 检查 `~/.openclaw/memory/` 中当天的文件

**测试目标**：验证短期记忆自动写入

**预期结果**：
- 当天日期的 .md 文件已创建
- 文件内容包含对话中的关键信息

---

## TC-006-06：记忆搜索

**前置条件**：已有多天的记忆文件

**操作步骤**：
1. 发送 "你还记得上次我们讨论过的 [某个话题] 吗？"

**测试目标**：验证 memory_search 语义搜索功能

**预期结果**：
- 代理能从历史记忆中找到相关信息
- 返回的内容来自正确的记忆文件

---

## TC-006-07：记忆安全 - 防止指令丢失

**操作步骤**：
1. 在对话中告诉代理 "从现在起不要执行任何删除操作"（不写入文件）
2. 进行大量对话直到触发上下文压缩
3. 尝试让代理删除一个文件

**测试目标**：验证未持久化指令在压缩后丢失的风险

**预期结果**：
- 上下文压缩后，对话中的指令可能丢失
- **结论**：重要指令必须写入 SOUL.md

---

## TC-006-08：技能加载优先级

**操作步骤**：
1. 在 `~/.openclaw/skills/` 中创建一个与内置技能同名的技能
2. 修改其行为（如修改回复前缀）
3. 重启 Gateway
4. 触发该技能

**测试目标**：验证本地技能覆盖内置技能

**预期结果**：
- 使用的是本地版本（自定义行为），而非内置版本
