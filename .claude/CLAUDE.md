# 项目说明

这是 OpenClaw AI 代理工具的中文配置和使用指南仓库。

## 项目结构

```
guide/          - 10 章配置与使用指南（中文）
testcase/       - 每章对应的测试用例
scripts/        - 安装、备份、健康检查脚本
.claude/skills/ - 推荐的 OpenClaw 技能模板
```

## 编写规范

- 所有指南使用中文撰写
- 技术术语保留英文（如 API Key、Gateway、Docker）
- 代码示例使用 JSON5 格式（支持注释）
- 每章末尾需包含参考来源链接
- 测试用例需包含：操作步骤、测试目标、预期结果

## 内容要求

- 所有技术内容必须有真实来源，标注参考链接
- 配置示例必须基于 OpenClaw 官方文档
- 安全建议需引用安全研究报告
- 保持时效性，标注版本和日期信息
